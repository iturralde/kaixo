Helburuak

git bertsio-kontroleko sistema banatua ikastea. github zerbitzuak erabiltzea. git eta github bidez proiektuak kudeatzea: sortu, eskuratu, aldatu eta eguneratu.

Edukiak

Hauek dira landuko diren gai nagusiak:

  1. Bertsio-kontroleko sistema banatuak: git

git-en erabilera eta oinarrizko eragiketak. git-en erabilera elkar-lanerako; gordailuak konpartitzen. Kode irekiko ohitura onak.

  1. github ostalaritza-zerbitzua

proiektuak sortu eta eskuratu. aldaketak egin eta konpartitu: pull request eredua.

Metodologia

Ikastaroa bi zati nagusitan banatuko da. Lehenengo zatian, git sistema aurkeztuko da, eta bere bidez buru daitezkeen eragiketa ohikoenak azalduko dira. Bigarren zatian github zerbitzua erakutsiko da, eta github bidez proiektuak kudeatzen erakutsiko da. Bi zatietan hainbat ariketa burutuko dira. Nori zuzendua

Informatikariei, kode irekiaz interesatuta dauden guztiei, software garapena hobetzeko interesatuta daudenei.
